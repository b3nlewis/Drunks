Welcome to Planning for Drunks's documentation!
===============================================

Website Description
-------------------

This website shows the documentation for Planning for Drunks.
It is an agent based model which simulates drunks returning home from the pub.
There are two methods to use, simple or advance. Advance does not allow backtracking and has a faster process time.
The documentation in this website details function descriptions, pararmeters and returns.
